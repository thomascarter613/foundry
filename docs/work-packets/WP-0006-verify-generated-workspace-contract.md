# WP-0006: Verify Generated Workspace Contract

## Purpose

Add a verifier for the workspace generated by the `foundry init` MVP path.

## Problem

The init smoke fixture confirms whether a non-interactive init invocation succeeds, but it does not yet verify the generated workspace as a product artifact.

Foundry MVP requires the generated workspace to have a minimum contract.

## Deliverables

This Work Packet delivers:

1. `tools/scripts/check-generated-workspace-contract.sh`
2. A generated workspace contract summary under `.artifacts/foundry/generated-workspace-contract/`
3. Verification that the init smoke workspace exists
4. Verification of required generated files
5. Verification of required generated directories
6. Verification of required package scripts
7. Verification of Foundry provenance/audit anchors when present

## Contract Under Test

The verifier checks the workspace generated at:

~~~text
.artifacts/foundry/init-workspace/workspace
~~~

Required directories:

- `apps`
- `services`
- `packages`
- `docs`
- `tools/scripts`
- `contracts/openapi`
- `generated/clients`
- `config/foundry`
- `templates`

Required files:

- `package.json`
- `bunfig.toml`
- `README.md`
- `.gitignore`
- `tsconfig.base.json`
- `turbo.json`
- `.github/workflows/ci.yml`
- `tools/scripts/foundry.sh`
- `tools/scripts/verify.sh`
- `packages/cli/src/index.ts`
- `config/foundry/generator-manifest.json`
- `.scaffdog/config.js`

Required scripts in generated `package.json`:

- `foundry`
- `verify`

## Verification

Run:

~~~bash
tools/scripts/check-foundry-init-workspace.sh
tools/scripts/check-generated-workspace-contract.sh
cat .artifacts/foundry/generated-workspace-contract/summary.json
~~~

## Acceptance Criteria

This Work Packet is accepted when:

1. Init workspace smoke succeeds.
2. Generated workspace contract verification succeeds.
3. Summary JSON reports `"ok": true`.
4. The generated workspace contains all required MVP contract directories.
5. The generated workspace contains all required MVP contract files.
6. The generated workspace exposes required baseline package scripts.
