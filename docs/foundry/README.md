# Foundry Project Index

Foundry is the first shippable product inside the broader AionX ecosystem.

Foundry is a repo-generation, repo-validation, repo-evolution, docs/spec/work-packet, AI-ready software delivery CLI.

It exists to help a developer or team create and evolve high-quality software repositories without repeatedly rewriting the same project structure, documentation system, verification scripts, generator manifests, governance files, and development workflow.

## Canonical Product Statement

Foundry helps users initialize, inspect, validate, generate, document, and evolve software repositories using a governed, reproducible, verification-first workflow.

## Foundry's Role in AionX

AionX is the umbrella system.

Foundry is the CLI and repository machinery layer.

Charon is the repo-native context and memory layer.

Workbench is the future visual interface.

Ardtire is a flagship reference implementation.
